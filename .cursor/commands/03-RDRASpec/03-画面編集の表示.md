# UIエディタサーバー起動

## 目的

画面で必要な項目を確認し、編集します

## 実施手順

ターミナルで以下の手順を実行してください：

### macOS / Linux

```bash
node RDRA_Knowledge/helper_tools/web_tool/ui_server.js
```

その後、ブラウザで http://localhost:3000 を開いてください。または、以下のコマンドで自動的に開けます：

```bash
open http://localhost:3000  # macOS
xdg-open http://localhost:3000  # Linux
```

### Windows

```cmd
node RDRA_Knowledge\helper_tools\web_tool\ui_server.js
```

その後、ブラウザで http://localhost:3000 を開いてください。または、以下のコマンドで自動的に開けます：

```cmd
start http://localhost:3000
```

ポート3000が既に使用されている場合は、既存のサーバーに接続してください。
