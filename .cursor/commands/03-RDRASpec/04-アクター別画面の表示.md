# アクターUI表示

## 目的

アクター別の画面を表示し、要件のすり合わせを行います

## 実施手順

ターミナルで以下の手順を実行してください：

### macOS / Linux

```bash
node RDRA_Knowledge/helper_tools/convertUI2ActorUI.js
node RDRA_Knowledge/helper_tools/web_tool/actorUI_server.js
```

その後、ブラウザで http://localhost:3001 を開いてください。または、以下のコマンドで自動的に開けます：

```bash
open http://localhost:3001  # macOS
xdg-open http://localhost:3001  # Linux
```

### Windows

```cmd
node RDRA_Knowledge\helper_tools\convertUI2ActorUI.js
node RDRA_Knowledge\helper_tools\web_tool\actorUI_server.js
```

その後、ブラウザで http://localhost:3001 を開いてください。または、以下のコマンドで自動的に開けます：

```cmd
start http://localhost:3001
```

前提条件: `/2_RDRASpec/ui.json` が存在すること。サーバーが既に起動している場合は、ブラウザで http://localhost:3001 にアクセスしてください。
