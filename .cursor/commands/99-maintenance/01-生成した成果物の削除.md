# 成果物の削除

## 目的

- 以下フォルダ下に生成されたファイルを削除する
  - `/0_RDRAZeroOne/`
  - `/1_RDRA/`
  - `/2_RDRASpec/`

## 実施手順

以下の手順に従って、ご利用のOSに応じたコマンドを実行してください：

### macOS / Linux

ターミナルで以下のコマンドを実行：

```bash
node RDRA_Knowledge/helper_tools/deleteFiles.js
```

### Windows

```bash
node RDRA_Knowledge/helper_tools/deleteFiles.js
```
